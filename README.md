# NetworkAnalysis

## Function & Features
### 1 WiFi scanning: 
	 * Get the WiFi SSID access points around the android device, incluing information: SSID, WiFi signal strength, channel and encryption method.
### 2 Device scanning: 
	* Scan all alive devices which under the wifi your android device is located and connected. 
	* Identify the device's manufacturer base on Mac address.
### 3 Speed test:
	* Test the upload and download speed under the current network connection through JSpeedTest.
### 4 Network diagnosis
	* Connectivity to gateway
	* Connectivity to DNS Server
	* Resovling Domain name(default:www.google.com)
	
### 5 Setting
	* Language support:
	* Theme
